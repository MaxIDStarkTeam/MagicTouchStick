# MagicTouchStick
Touch Block to get that Block,Only Pocketmine-MP

# ⚙️Commands
| Command | Permission | Default |
| --- | --- | --- |
| `/mstick` | `magic.stick` | **op** |

# 🛡️Features
 - If u Touch Block,you will get the block that you touch
 - Only Op
